Privacy Policy

This Privacy Policy defines the rules for collecting, processing, and using the personal data of users of the SayItAgain mobile application, created and provided by TomatoGaming.

1. Data Controller

Name: TomatoGaming

Address: Katowice, Poland

Contact (e-mail): szyjanq@gmail.com

2. Scope and purpose of data collection

  • Internet access: required to play audio tracks.
  • Camera access: required to scan QR codes in physical mode.

No permanent storage of user data: The application does not permanently store any data entered by the user (nicknames, answers, results). All data is stored only in working memory and deleted after the application is closed.

3. Legal basis for processing

Processing is necessary for the performance of the service in accordance with the contract with the user (Article 6(1)(b) GDPR), i.e. the operation of the QR scanner and playback of quotes.

4. Data sharing

The Controller does not share any collected data with third parties.

5. Data retention period

Data is processed only during the user's session and deleted immediately after it ends (closing the application).

6. User rights

The user has the right to:

  • access their data,
  • rectify data,
  • erase data ("right to be forgotten"),
  • restrict processing,
  • object to processing,
  • data portability,
  • lodge a complaint with the President of the Personal Data Protection Office.

7. Changes to the Privacy Policy

The Controller reserves the right to make changes to the Privacy Policy. Users will be informed of any changes via an updated version of the document available in the application and on the website.
